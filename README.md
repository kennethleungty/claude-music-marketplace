# code-music marketplace

A Claude Code plugin marketplace for [code-music](https://github.com/kennethleungty/code-music) — background music for your AI coding agent sessions.

## Installation

In Claude Code, register the marketplace:

```
/plugin marketplace add kennethleungty/code-music-marketplace
```

Then install the plugin:

```
/plugin install code-music@code-music-marketplace
```

## Available Plugins

### code-music

Background music for your coding sessions — lofi, jazz, classical, ambient, electronic, and more. Streaming live from the internet, right in your terminal.

```
/plugin install code-music@code-music-marketplace
```

No setup, no accounts, no ads. Just install and play.

**Commands:** `/play`, `/stop`, `/vibe`, `/mood`, `/focus`, `/volume`, `/next`, `/status`, and more.

[Full documentation →](https://github.com/kennethleungty/code-music)
